# Privacy Policy

**Last updated:** March 6, 2026

## Overview

Fan Facts Drop ("we", "our", "the Service") is an automated content creation and publishing platform. This policy describes how we handle data in connection with our use of third-party APIs, including the TikTok API.

## Data We Collect

- **Video metadata:** Topics, facts, image URLs, and rendering parameters used to produce short-form videos.
- **API tokens:** OAuth tokens required to publish content to platforms (TikTok, YouTube, etc.). Tokens are stored securely in Azure and are never shared with third parties.
- **Publishing logs:** Timestamps, platform post IDs, and publishing status for operational monitoring.

## Data We Do NOT Collect

- We do **not** collect personal data from viewers or end users.
- We do **not** track, profile, or identify any individuals.
- We do **not** use cookies or analytics on any viewer-facing content.

## How We Use Data

All data is used solely to:
1. Generate and render short-form educational videos.
2. Publish videos to social media platforms via their official APIs.
3. Monitor publishing status and troubleshoot errors.

## Data Storage

- Data is stored in Microsoft Azure (West Europe region).
- API credentials are stored as encrypted environment variables and are never committed to source code.

## Data Retention

- Video metadata and publishing logs are retained for up to 90 days for operational purposes.
- OAuth tokens are retained only as long as they are needed for publishing and can be revoked at any time.

## Third-Party Services

We use the following third-party APIs:
- **TikTok Content Posting API** — for publishing videos
- **YouTube Data API** — for publishing videos
- **Azure OpenAI** — for content generation
- **Azure Cognitive Services** — for text-to-speech

We comply with each platform's API terms of service and developer policies.
